  \item Run a context-aware generation pass to align imports, aliases, and utility helpers.
  \item Apply semantic token remapping and validate density or motion variants.
  \item Execute patch diff review and preserve product-specific instrumentation.
  \item Publish only after passing type, accessibility, interaction, and visual gates.
\end{enumerate}
\textbf{Reference command flow.}
\begin{verbatim}
ui-tool init --preset enterprise-core
ui-tool add data-grid tenant-switcher
ui-tool sync --brand default --plan

ui-tool doctor --strict
\end{verbatim}
\textbf{Quality gates.}
\begin{itemize}[leftmargin=*, itemsep=0.24em]
  \item Type-check generated source and fail on unresolved alias paths.
  \item Run accessibility tests for keyboard flow, focus visibility, and label integrity.
  \item Compare visual snapshots across compact, comfortable, and spacious density profiles.
  \item Reject patches that increase bundle size beyond agreed thresholds.
\end{itemize}

\begin{itemize}[leftmargin=*, itemsep=0.24em]
  \item Patch-merge success rate above 95% across three consecutive releases.
  \item Accessibility escape defects reduced below one per release cycle.
  \item Mean review time for shared primitive updates below one business day.
\end{itemize}
